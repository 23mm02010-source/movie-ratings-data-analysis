# movie-ratings-data-analysis
Exploratory Data Analysis on Fandango movie ratings dataset using Python, pandas, Matplotlib and Seaborn
# 🎬 Movie Ratings Data Analysis

## Project Overview

This project analyzes movie ratings from the Fandango website and compares them with ratings from other movie review platforms. The goal is to examine whether Fandango ratings are biased.

## Problem Statement

Fandango is a movie ticket selling platform that also publishes movie ratings. Since they benefit financially from ticket sales, there is a possibility that the ratings displayed on the platform may be higher than ratings from other review sites.

## Analysis

Using Python and data analysis libraries, the ratings from Fandango were compared with ratings from other movie review websites. Scatter plots and correlation analysis were used to observe the relationship between ratings and number of votes.

## Key Observation

The analysis shows that **Fandango movie ratings tend to be consistently higher than ratings from other review platforms**.

## Conclusion

Since Fandango also sells movie tickets, this suggests a possible bias where ratings might be inflated to encourage more ticket purchases.
